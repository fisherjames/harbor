# @harbor/api

## Purpose
tRPC router contracts for workflow save/deploy/publish and run operations.

## Entrypoints
- `src/router.ts`: typed router and enforcement logic.
- `src/types.ts`: stable API contracts shared with callers.
- `src/index.ts`: public package exports.

## Commands
- `pnpm --filter @harbor/api typecheck`
- `pnpm --filter @harbor/api test`
